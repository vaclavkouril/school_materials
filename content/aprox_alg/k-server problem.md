# $k$-server problem
- $M$ a metric space
- $r_{1},\dots,r_{n}\in M$
- objective is to minimize distance traveled by all servers.

### *Vﾄ孚a:* For a tree metric, there exists a $k$-competitive deterministic algorithm.

### *Vﾄ孚a:* For any $M$, $(2k-1)$-competitive deterministic algorithm.

### Vﾄ孚a: Lower bound $\Omega(\log^2 k)$ for randomized algorithm.
